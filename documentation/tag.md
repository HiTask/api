Tag
========

Operations with tags: add, remove, list tags


Get list of user's tags
------------

* `GET 	/tag` will return list of user's tags

### Example response

```js
{

}
```

    <tr>
        <th>Field</th>

        <td><code>id</code></td>
        <td>long</td>
        <td>Globally unique primary identifier for a tag.</td>
    </tr>
 
</table>



Add tag
------------

* `POST	/tag`


Delete
------------

* `DELETE	/tag` 


List tags
------------

* `GET		/tag`
